Code LocationsÂ¶

ownCloud has the ability to find parts of its code in non-standard locations. This section describes how to configure ownCloud for that functionality.

Parameter Format Description
Theme “theme” => “”, If the instance of ownCloud is themed, the name of the theme should be configured here. For more information on this parameter, see the document on Theming.
3 rd ** Party Root** “3rdpartyroot” => “”, ownCloud uses some 3 rd party PHP components to provide certain functionality. These components are shipped as part of the software package and reside in ~/owncloud/3rdparty. However, if this folder resides elsewhere, the location can be configured here. For example /srv/http/path/to/3rdparty.
3 rd ** Party URL** “3rdpartyurl” => “”, In the event that the 3rdpartyroot is configured, this parameter should be configured as well to show the http web path to the 3rdpartyroot starting at the owncloud web root. For instance /path/to/3rdparty.
