A plugin which Enable Network admin/Super admin to manage the Plugins for Individual Blog in the Multi site network.

== Description ==

In WordPress Network/WPMU we have option for Enabling plugins for network wide but we don't have option to customize it for some specific Blog/site 

so with the help of this plugin you can have that feature.so if you want to enable plugin in some site not throughout the network then with the help of this plugin you can do that very easily 

== Installation ==

this plugin is for WPMU/WP Network so to use this you must have WPMU and the GUI of the Plugin will be only visible to the Super Admin at Network Menu. 

1. Upload `advanced-wpmu-pluginmanager` folder to the `/wp-content/plugins/` directory
2. Activate the plugin for network wide through the 'Plugins' menu in WordPress
3.On the Network Dashboard YOu will see the Option Names Plugin Manager From there You Can Control the Rest Plugins for Individual Blog
